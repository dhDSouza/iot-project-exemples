# iot-project-exemples 🌐

### Exemplos de projetos com eletrônica e internet das coisas.

_**OBS:** Este repositório também será utilizado para a confecção das atividades de GIT relacionadas ao estágio na COMPASS UOL._

# Lista de atividades GIT 📖👨‍💻

1. Criar uma conta grátis no GitHub (caso não possua)

2. Criar um repositório de exemplo na sua conta

3. Praticar seus conhecimentos neste repositório:
  - Criar branches, fazer commits, merge entre branches, descrever as alterações feitas nas mensagens de commit, etc
  
4. Enviar o link do seu repositório no GitHub conforme solicitado
- [Link do repositório](https://github.com/DaniEUgit/iot-project-exemples)

# Lista de atividades Markdown 📖📑

## Após a conclusão do material de apoio, completar as seguintes tarefas:

1. Crie um arquivo chamado 'Tutorial.md' no seu repositório público do Git Hub criado no tópico anterior

2. Escreva um texto de exemplo neste arquivo

- Sugestão: pratique os comandos de markdown ensinados nos tutoriais para criar o arquivo

3. Faça o commit e o push deste arquivo e quais outros arquivos que precise para demonstrar o seu exemplo

4. Informe a URL do arquivo conforme solicitado:

- [Link do Tutorial.md](https://github.com/DaniEUgit/iot-project-exemples/blob/main/envio-dados-thingspeak/Tutorial.md)

5. Assegure-se de demonstrar tudo que aprendeu: cabeçalhos, negrito, itálico, listas, tabelas, bloco de código, imagens, etc.
